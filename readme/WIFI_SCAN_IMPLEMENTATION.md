## 🎯 功能特性

### 核心功能

1. **自动扫描**
   - 一键扫描周围所有WiFi热点
   - 自动开启WiFi（如果未开启）
   - 扫描过程显示状态提示

2. **详细信息展示**
   - WiFi名称（SSID）
   - 信号强度（dBm + 等级：优秀/良好/一般/差）
   - 频段识别（2.4G / 5G）
   - 加密方式（WPA2 / WPA3 / WPA / Open）
   - MAC地址（BSSID）

3. **智能排序**
   - 按信号强度从强到弱自动排序
   - 方便用户选择信号最好的WiFi

4. **便捷连接**
   - 双击WiFi即可快速连接
   - 自动识别加密方式
   - 开放WiFi无需输入密码

5. **历史记录**
   - 自动填充上次使用的密码
   - 连接成功后保存配置

## 📱 使用方法

### 方法 1：通过菜单扫描连接（新功能）

1. **打开工具**
   ```
   工具 → 🔍 扫描并连接WiFi
   ```

2. **选择设备**（如果有多个激活盒子）
   - 从下拉列表选择要扫描WiFi的设备
   - 点击"确定"

3. **扫描WiFi**
   - 点击"🔄 扫描WiFi"按钮
   - 等待3-5秒
   - 列表显示所有可用WiFi

4. **选择并连接**
   - 双击要连接的WiFi，或
   - 单击选中WiFi + 输入密码 + 点击"连接选中WiFi"

5. **等待连接**
   - 系统自动执行连接和网络测试
   - 显示连接结果和网络连通性

### 方法 2：手动输入SSID连接（原有功能）

1. **打开工具**
   ```
   工具 → 激活盒子WiFi链接
   ```

2. **输入信息**
   - 选择设备
   - 手动输入SSID
   - 输入密码
   - 选择加密方式（wpa2/wpa3）

3. **点击连接**
   - 系统执行连接流程

## ⚠️ 注意事项

### 1. 设备要求
- ✅ 设备必须支持 `cmd wifi` 命令（Android 8.0+）
- ✅ 设备WiFi硬件必须正常工作
- ✅ ADB连接必须正常

### 2. 扫描限制
- ⏱ 扫描需要3-5秒时间
- 📡 扫描结果受设备位置和信号强度影响
- 🔄 可能需要多次扫描才能发现所有WiFi

### 3. 连接注意
- 🔐 加密WiFi必须输入正确密码
- 🌐 开放WiFi可能不安全，谨慎连接
- ⏳ 连接后网络测试需要额外时间（约10-20秒）

### 4. 已知问题
- ❗ 某些设备可能不支持 `cmd wifi` 命令，请使用手动输入方式
- ❗ 中文SSID可能显示异常，功能不受影响
- ❗ 隐藏SSID的WiFi无法通过扫描发现，请使用手动输入方式

## 🔧 故障排查

### 问题 1：扫描返回空列表

**可能原因**：
- WiFi未开启
- 设备不支持扫描命令
- 等待时间不足

**解决方法**：
1. 手动测试ADB命令（见上方测试命令）
2. 检查设备Android版本
3. 尝试手动输入连接方式

### 问题 2：无法解析扫描结果

**症状**：扫描完成但列表为空

**解决方法**：
1. 查看日志文件 `logs/cbss_tool.log`
2. 检查ADB命令输出格式
3. 向开发者反馈设备型号和Android版本

### 问题 3：连接失败

**可能原因**：
- 密码错误
- 加密方式不匹配
- 信号太弱

**解决方法**：
1. 确认密码正确
2. 尝试选择信号更强的WiFi
3. 使用手动输入方式尝试连接
